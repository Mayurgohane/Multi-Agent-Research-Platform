# Frontend for MARP — Multi-Agent Academic Research Platform

Next.js App Router UI for launching research jobs, watching agent progress, and reading
cited literature reviews.

## Develop

```bash
cp .env.example .env.local
npm install
npm run dev
```

Open http://localhost:3000

Requires the backend API at `NEXT_PUBLIC_API_BASE_URL` (default `http://localhost:8000`).

## Build

```bash
npm run build
npm start
```
